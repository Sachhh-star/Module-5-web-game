Mini Math Game

Welcome to the Mini Math Game!
This project is an interactive game where you solve math problems, earn points, and level up - all while racing against a timer.
The goal is to make practicing math both fun and challenging!

How Does It Work?

  • ➜ Solve random math questions with operators like +, -, ×, and ÷.
  • ⌛ Type in your answer before the timer runs out.
  • ⭐ Earn points with every correct answer and unlock new levels.
  • 🚀 The questions get harder as you progress!
  • 💡 Made with HTML, CSS & JavaScript.
